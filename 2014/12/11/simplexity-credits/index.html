credits, components

Simplexity is build with the help of the following open source tools and libraries:

  • Jekyll - great static site generator with some great feature
  • Basscss - a great low level CSS toolkit
  • reworknpm - great tool to help keep basscss modules organized and easily managed
  • reworkcss - basscss is built using rework a CSS preprocessor
  • Node.js - I think when they said ‘Java everywhere’ they really meant Javascript
  • gulp.js - task runner with nice slick API and great streaming features
  • Fontawesome - glyphs and icons, as the name says pretty awesome

Many thank to all those who contributed to these awesome projects!

Simplexity has been inspired by Pixyll, a great Jekyll theme by John Otander (@4lpine). Many thanks for the inspiration!

Powered by Simplexity

Simplexity is a minimalistic theme for Jekyll. It is ment to showcase your content, using a monochrome color scheme and a fluid and responsive design. Large, legible fonts scale well across devices and various viewport sizes. Simplexity is licensed under MIT license.
